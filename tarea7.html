01 · Introducción

¿Qué es Arduino?

Arduino es una plataforma de electrónica de código abierto basada en hardware y software fáciles de usar. Fue creada en 2005 en el Instituto de Diseño Interactivo de Ivrea, Italia, por Massimo Banzi y su equipo, con el objetivo de que cualquier persona —sin conocimientos previos de electrónica— pudiera crear proyectos tecnológicos interactivos.

En términos técnicos, Arduino es una placa microcontroladora: un pequeño computador que puede leer señales del entorno (sensores, botones, temperatura) y actuar sobre él (encender LEDs, mover motores, enviar datos). Su programación se realiza en un lenguaje basado en C/C++ simplificado, desde un entorno de desarrollo gratuito llamado Arduino IDE.

Arduino se ha convertido en el punto de entrada mundial para la robótica, domótica, IoT (Internet de las Cosas), arte interactivo y prototipado electrónico. Millones de estudiantes, ingenieros y makers lo usan en todo el mundo.

⚡ Concepto clave

Arduino no es solo una placa: es un ecosistema completo de hardware, software y comunidad. La placa es el cerebro, los componentes son los sentidos y las manos, y el código es el pensamiento que los conecta.

03 · Pines y conexiones

Pines y tipos de conexión

Los pines son los puntos de conexión entre la placa Arduino y los componentes externos. Cada pin tiene una función específica.

Tipos de pines — Arduino Uno

5V
5V / 3.3V Pines de alimentación. Suministran voltaje a los componentes conectados.
GND
GND Tierra (Ground). Referencia de voltaje cero. Todos los circuitos necesitan al menos uno.
D
Digital (0-13) Leen o escriben valores digitales: HIGH (5V) o LOW (0V). Ideales para LEDs y botones.
~
PWM (3,5,6,9,10,11) Modulación de Ancho de Pulso. Simulan salidas analógicas (0-255). Para dimmer de LEDs y velocidad de motores.
A
Analógico (A0-A5) Leen voltajes entre 0V y 5V con resolución de 10 bits (valores 0-1023). Para sensores de temperatura, luz, etc.
TX/RX
Serial (0 y 1) Transmisión y recepción de datos serial. Usados para comunicación con PC y módulos.
I²C
I2C (A4-SDA, A5-SCL) Bus de comunicación para conectar múltiples dispositivos con solo 2 cables (datos y reloj).
SPI
SPI (10-13) Bus de alta velocidad para tarjetas SD, pantallas TFT y módulos de comunicación rápida.
04 · IDE y programación

Arduino IDE: estructura básica del código

Todo programa de Arduino (llamado sketch) tiene dos funciones obligatorias: setup() que se ejecuta una sola vez al encender, y loop() que se repite indefinidamente.

blink.ino
// Estructura base de todo sketch Arduino

// setup() — se ejecuta UNA SOLA VEZ al encender
void setup() {
  pinMode(13, OUTPUT);      // pin 13 como salida (LED integrado)
  Serial.begin(9600);       // iniciar comunicación serial
  Serial.println("¡Hola!"); // mensaje inicial
}

// loop() — se repite CONTINUAMENTE
void loop() {
  digitalWrite(13, HIGH);  // encender LED
  delay(1000);              // esperar 1 segundo
  digitalWrite(13, LOW);   // apagar LED
  delay(1000);              // esperar 1 segundo
}

// Funciones más usadas:
// pinMode(pin, MODE)       → configura pin (INPUT/OUTPUT)
// digitalWrite(pin, val)  → escribe HIGH o LOW
// digitalRead(pin)        → lee HIGH o LOW
// analogRead(pin)         → lee 0-1023 (pines A0-A5)
// analogWrite(pin, val)   → PWM: escribe 0-255
// delay(ms)               → pausa en milisegundos
// millis()                → tiempo desde encendido (ms)

06 · Componentes Pasivos

Componentes Pasivos

Los componentes pasivos no requieren energía propia para funcionar. Son los bloques de construcción fundamentales de cualquier circuito electrónico.

🟫
Resistencia
Pasivo

Limita el flujo de corriente eléctrica en un circuito. Indispensable para proteger LEDs y otros componentes de sobrecargas. Se mide en Ohmios (Ω) y su valor se identifica por bandas de color.

Ohmios ΩProtege LEDsBandas de color
⚪
Capacitor
Pasivo

Almacena y libera energía eléctrica. Se usa para filtrar ruido en la alimentación, estabilizar voltajes y crear retardos en circuitos temporizadores. Se mide en Faradios (F).

Faradios FFiltro ruidoElectrolítico/Cerámico
🌀
Inductor / Bobina
Pasivo

Almacena energía en un campo magnético. Se usa en fuentes de poder conmutadas, filtros y motores. Resiste cambios bruscos de corriente. Se mide en Henrios (H).

Henrios HCampo magnéticoFiltro AC
📏
Potenciómetro
Pasivo / Entrada

Resistencia ajustable manualmente mediante una perilla o deslizador. Permite controlar el voltaje de entrada a un pin analógico. Ideal para controlar brillo, volumen y velocidad.

VariableA0-A50-1023
🔘
Pulsador / Botón
Entrada digital

Interruptor momentáneo que cierra o abre un circuito al presionarlo. Uno de los componentes de entrada más simples. Siempre debe usarse con una resistencia pull-up o pull-down para evitar lecturas erráticas.

DigitalPull-upINPUT_PULLUP
🔲
Protoboard
Herramienta

Placa de prototipado con conexiones internas que permiten armar circuitos sin soldar. Las filas centrales están conectadas horizontalmente; las columnas de los extremos son los rieles de alimentación (+/-). Esencial para hacer pruebas.

Sin soldaduraPrototipado830 orificios
07 · Sensores

Sensores: los sentidos de Arduino

Los sensores convierten magnitudes físicas del entorno (temperatura, luz, distancia, movimiento) en señales eléctricas que Arduino puede leer y procesar.

🌡️
Sensor de Temperatura DHT11 / DHT22
Sensor ambiental

Mide temperatura y humedad relativa del ambiente. El DHT22 tiene mayor precisión (±0.5°C). Se comunica con Arduino por un solo pin digital. Muy usado en estaciones meteorológicas caseras.

-40°C a 80°CDigitalHumedad1 pin
📡
Sensor Ultrasónico HC-SR04
Sensor de distancia

Mide la distancia a objetos emitiendo ondas ultrasónicas y midiendo el tiempo de eco. Rango de 2cm a 400cm con precisión de 3mm. Ideal para robots, alarmas de proximidad y detectar obstáculos.

2-400 cmTrigger + Echo5V
👁️
Sensor de Luz LDR
Sensor de luz

Resistencia que cambia su valor según la intensidad de luz (LDR = Light Dependent Resistor). Con un divisor de voltaje conectado a un pin analógico, permite medir luminosidad ambiental. Útil en encendido automático de luces.

Analógico A0-A50-1023Divisor voltaje
🔴
Sensor Infrarrojo (IR)
Sensor de proximidad

Detecta objetos o superficies cercanas usando luz infrarroja invisible al ojo humano. Existen módulos para seguir líneas (robots de línea), evitar obstáculos, y receptores IR para controles remotos.

Digital2-30 cmSeguidor de línea
🏃
Sensor PIR de Movimiento
Sensor de movimiento

Detecta el movimiento de personas o animales midiendo la radiación infrarroja que emiten sus cuerpos. Salida digital: HIGH cuando detecta movimiento. Usado en alarmas, luces automáticas y sistemas de seguridad.

Digital3-7 metros5V-12V
📦
Acelerómetro / Giroscopio MPU-6050
Sensor de movimiento

Módulo que combina acelerómetro y giroscopio en 6 ejes. Mide inclinación, vibración y rotación. Comunicación I2C. Fundamental en drones, robots de equilibrio y proyectos de estabilización.

I2C6 ejes3.3V-5V
🌿
Sensor de Humedad de Suelo
Sensor ambiental

Mide la conductividad del suelo para determinar el nivel de humedad. Salida analógica o digital. Perfecta para sistemas de riego automático y proyectos de agricultura inteligente.

Analógico3.3V-5VAgricultura
💧
Sensor de Lluvia / Agua
Sensor ambiental

Detecta la presencia de agua sobre su superficie conductora. Salida analógica (cantidad de agua) y digital (¿hay agua?). Útil en sistemas de riego, alertas de inundación y toldos automáticos.

Digital + Analógico5VLluvia

Sensor de Sonido / Micrófono
Sensor de audio

Detecta intensidad sonora del ambiente. Salida digital para umbrales de sonido (aplausos, golpes) o analógica para nivel de audio. Se usa en encender luces con aplausos y proyectos de control por voz.

08 · Actuadores

Actuadores: las manos de Arduino

Los actuadores convierten las señales eléctricas de Arduino en acciones físicas: movimiento, sonido, calor. Son la forma en que Arduino interactúa con el mundo.

⚙️
Motor DC
Actuador de movimiento

Motor de corriente continua que convierte energía eléctrica en rotación. Para controlarlo desde Arduino se necesita un driver como el L298N (no conectar directo). Permite controlar sentido y velocidad (PWM).

Driver L298NPWM velocidadBidireccional
🎯
Servomotor SG90 / MG996R
Actuador de posición

Motor con control de posición angular precisa (0° a 180°). Tiene 3 cables: VCC, GND y señal PWM. El SG90 es pequeño y económico; el MG996R tiene mayor torque y es metálico. Esencial en brazos robóticos.

0°-180°PWM3 cables#include Servo.h
🔄
Motor a Pasos (Stepper)
Actuador de precisión

Motor que gira en pasos discretos y precisos. No necesita retroalimentación para saber su posición exacta. Requiere un driver como el ULN2003 o A4988. Usado en impresoras 3D, CNC y plotters.

ULN2003A4988Pasos precisos
💡
Relé
Actuador de control

Interruptor electromagnético que permite a Arduino controlar cargas de alto voltaje (220V, lámparas, electrodomésticos) usando una señal de 5V. Actúa como puente entre circuitos de baja y alta potencia.

5V control220V cargaNO / NC / COM
🔔
Buzzer / Zumbador
Actuador de sonido

Genera sonido a partir de señales eléctricas. El buzzer pasivo requiere una señal PWM (puede generar melodías); el activo solo necesita 5V para sonar a una frecuencia fija. Ideal para alertas y efectos de sonido.

Activo/Pasivotone()Melodías
💧
Bomba de Agua Mini
Actuador de fluidos

Bomba peristáltica o centrífuga de bajo voltaje para mover líquidos. Se controla mediante un transistor o módulo relé. Fundamental en sistemas de riego automático, fuentes y proyectos de acuaponía.

3V-6VTransistor/ReléRiego auto

14 · Ejemplos de Código

Ejemplos prácticos de componentes

sensor_distancia.ino
// HC-SR04: leer distancia en centímetros
const int TRIG = 9;
const int ECHO = 10;

void setup() {
  pinMode(TRIG, OUTPUT);
  pinMode(ECHO, INPUT);
  Serial.begin(9600);
}

void loop() {
  // Enviar pulso ultrasónico
  digitalWrite(TRIG, LOW);
  delayMicroseconds(2);
  digitalWrite(TRIG, HIGH);
  delayMicroseconds(10);
  digitalWrite(TRIG, LOW);

  // Medir tiempo de eco
  long duracion = pulseIn(ECHO, HIGH);

  // Calcular distancia: d = (t × velocidad_sonido) / 2
  float distancia = duracion * 0.034 / 2;

  Serial.print("Distancia: ");
  Serial.print(distancia);
  Serial.println(" cm");
  delay(500);
}
servo_motor.ino
// Servomotor: barrer de 0° a 180° y regresar
#include <Servo.h>

Servo miServo;

void setup() {
  miServo.attach(9);    // pin PWM del servo
}

void loop() {
  for (int ang = 0; ang <= 180; ang++) {
    miServo.write(ang);
    delay(15);
  }
  for (int ang = 180; ang >= 0; ang--) {
    miServo.write(ang);
    delay(15);
  }
}
dht_temperatura.ino
// DHT11: leer temperatura y humedad
#include <DHT.h>

#define PIN_DHT  2
#define TIPO_DHT DHT11

DHT dht(PIN_DHT, TIPO_DHT);

void setup() {
  Serial.begin(9600);
  dht.begin();
}

void loop() {
  float humedad     = dht.readHumidity();
  float temperatura = dht.readTemperature();

  if (isnan(humedad) || isnan(temperatura)) {
    Serial.println("Error de lectura");
    return;
  }

  Serial.print("Temp: ");
  Serial.print(temperatura);
  Serial.print("°C  Humedad: ");
  Serial.print(humedad);
  Serial.println("%");
  delay(2000);
}
15 · Buenas Prácticas

Buenas prácticas con Arduino

🛡️
Usa siempre resistencias
Protección

Nunca conectes un LED directamente a un pin de Arduino sin resistencia en serie. Sin ella, la corriente puede destruir el LED y dañar el pin del microcontrolador permanentemente.

⚡
Respeta la corriente
Seguridad

Cada pin de Arduino puede suministrar máximo 40mA y el total de todos los pines no debe exceder 200mA. Para motores y cargas grandes, usa siempre un driver externo como el L298N.

🔌
Voltaje 3.3V vs 5V
Compatibilidad

Muchos módulos modernos (ESP8266, RFID, MPU-6050) operan a 3.3V. Conectarlos directamente a pines de 5V puede dañarlos. Usa un divisor de voltaje o convertidor de nivel lógico.

📝
Comenta tu código
Legibilidad

Agrega comentarios a tu sketch explicando para qué sirve cada sección y a qué pin conectaste cada componente. Cuando vuelvas al proyecto después de días, lo agradecerás enormemente.

Prototipado

Siempre arma y prueba el circuito en una protoboard antes de soldarlo. Esto permite detectar y corregir errores sin dañar componentes ni placas de circuito impreso definitivas.

🌐
Usa librerías
Eficiencia

No reinventes la rueda. El ecosistema Arduino tiene librerías para casi todo: DHT.h, Servo.h, Wire.h, SD.h, LiquidCrystal.h. Instálalas desde el Gestor de Librerías del IDE.

🚀 Siguiente paso

Con estos componentes puedes construir robots, estaciones meteorológicas, sistemas de riego automático, alarmas inteligentes y mucho más. El siguiente nivel es combinar Arduino con Raspberry Pi para proyectos que requieran mayor procesamiento, o explorar el mundo del IoT con plataformas como Blynk y ThingSpeak.
