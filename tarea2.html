🌐 Desarrollo Web

Lenguajes para la Web

La Web funciona con una combinación de lenguajes que trabajan en capas. HTML da estructura, CSS da estilo, JavaScript da interactividad y lenguajes como PHP gestionan la lógica del servidor. Dominarlos es el primer paso de cualquier desarrollador.

HTML 5
Front-end · Marcado

HyperText Markup Language es el esqueleto de toda página web. Define la estructura y el significado semántico del contenido: encabezados, párrafos, listas, imágenes, formularios y más. Sin HTML, la Web no existe.

index.html
<!-- Estructura básica HTML5 -->
<header class="hero">
  <h1>Hola, Mundo</h1>
  <p>Bienvenido a mi sitio</p>
  <a href="#">Ver más</a>
</header>
CSS 3
Front-end · Estilos

Cascading Style Sheets controla la apariencia visual: colores, tipografías, espaciado, animaciones y el diseño responsivo. Es el lenguaje que convierte una estructura básica en una interfaz atractiva.

styles.css
.hero {
  display: flex;
  background: linear-gradient(
    135deg, #e8620a, #c94a00
  );
  border-radius: 16px;
  animation: fadeIn 0.6s ease;
}
JavaScript
Front/Back · Lógica

JavaScript es el único lenguaje nativo del navegador. Agrega interactividad a las páginas web: formularios dinámicos, animaciones, llamadas a APIs y hoy también corre en servidores con Node.js. Es el lenguaje más usado del mundo.

app.js
const saludar = (nombre) => {
  const msg = `Hola, ${nombre}!`;
  console.log(msg);
  return msg;
};

saludar("Matias"); // Hola, Matias!
PHP
Back-end · Servidor

PHP es el lenguaje de servidor que impulsa el 77% de la Web, incluyendo WordPress. Gestiona bases de datos, sesiones de usuario, formularios y genera HTML dinámico. Ideal para proyectos web clásicos y sistemas de contenido.

index.php
<?php
$nombre = "Matias";
$año    = 2025;

if ($año > 2000) {
  echo "Bienvenido, $nombre";
}
?>
🖥️ Aplicaciones de Escritorio

Lenguajes para Escritorio

Las aplicaciones que instalas en tu computadora — desde editores de imagen hasta software de ingeniería — están escritas en lenguajes de mayor rendimiento. Cada uno tiene sus fortalezas y ecosistema propio.

Python
IA · Ciencia de Datos

Python es el lenguaje favorito de la inteligencia artificial y el análisis de datos. Su sintaxis simple y su gigante ecosistema de librerías (TensorFlow, PyTorch, NumPy) lo hacen imprescindible para quien estudia IA y cultura digital.

ia_modelo.py
# Red neuronal simple
import tensorflow as tf

modelo = tf.keras.Sequential([
  tf.keras.layers.Dense(128,
    activation='relu'),
  tf.keras.layers.Dense(10)
])
modelo.compile(optimizer='adam')
Java
Multiplataforma · OOP

Java sigue el principio "escribe una vez, corre en cualquier lugar". Es la base de Android y miles de aplicaciones empresariales. Orientado a objetos, robusto y con más de 25 años de ecosistema consolidado.

Main.java
public class Main {
  public static void main(
      String[] args) {

    int x = 10;
    System.out.println(
      "Valor: " + x
    );
  }
}
C#
Windows · Videojuegos

C# (C Sharp) es el lenguaje de Microsoft, creador del ecosistema .NET. Es el lenguaje principal de Unity para videojuegos y de aplicaciones empresariales Windows. Potente, moderno y con tipado estricto.

Player.cs
public class Player
    : MonoBehaviour {

  public float velocidad = 5f;

  void Update() {
    float h = Input.GetAxis(
      "Horizontal");
    transform.Translate(
      h * velocidad);
  }
}
C++
Sistemas · Alto rendimiento

C++ es el lenguaje del rendimiento extremo. Opera a nivel de hardware, ideal para motores de videojuegos (Unreal Engine), sistemas operativos, software de simulación y aplicaciones que exigen velocidad máxima.

main.cpp
// Suma de vectores
#include <iostream>
#include <vector>
using namespace std;

int main() {
  vector<int> nums = {1,2,3};
  for (auto n : nums)
    cout << n << " ";
}
🎨 Diseño y Maquetación

Lenguajes y Herramientas de Diseño

El diseño digital también tiene su propio lenguaje. SCSS y Tailwind potencian el CSS clásico, mientras que herramientas como Figma y Adobe XD permiten prototipar interfaces antes de programarlas.

SCSS / Sass
Pre-procesador CSS

SCSS es un superconjunto de CSS que añade variables, anidamiento, mixins y funciones. Permite escribir hojas de estilo más organizadas, reutilizables y mantenibles. Se compila a CSS estándar para el navegador.

styles.scss
$color-prim: #e8620a;
$radio: 12px;

@mixin tarjeta($sombra: true) {
  border-radius: $radio;
  @if $sombra {
    box-shadow: 0 8px 24px
      rgba(0,0,0,.2);
  }
}
.card { @include tarjeta(); }
Tailwind
Framework CSS · Utilidades

Tailwind CSS es el framework de diseño más popular actualmente. En lugar de escribir CSS, aplicas clases directamente en el HTML. Permite construir interfaces complejas muy rápido sin salir del archivo HTML.

card.html
<div class="
  bg-white rounded-2xl
  shadow-lg p-6
  flex flex-col gap-4
  hover:scale-105
  transition-transform">
  <h2 class="text-xl font-bold">
    Tarjeta
  </h2>
</div>
Figma
UI/UX · Prototipado

Figma es la herramienta estándar de la industria para diseño de interfaces. Permite crear wireframes, prototipos interactivos y sistemas de diseño colaborativos en tiempo real. Su lenguaje interno incluye Auto Layout, Variables y componentes reutilizables.

componente.fig (export CSS)
/* CSS generado por Figma */
.btn-primario {
  display: flex;
  padding: 12px 24px;
  background: #A259FF;
  border-radius: 100px;
  font-weight: 600;
  color: #FFFFFF;
}
Xd
Adobe XD
UI/UX · Adobe Creative

Adobe XD es la herramienta de prototipado del ecosistema Adobe. Permite diseñar flujos de usuario, animar transiciones entre pantallas y exportar especificaciones de diseño para desarrolladores con integración directa a Photoshop e Illustrator.

pantalla.xd (export CSS)
/* Especificaciones XD */
.pantalla-inicio {
  width: 375px;
  height: 812px;
  background: #FF0099;
  font-family: "Adobe Clean";
  transition: all 0.3s
    cubic-bezier(.4,0,.2,1);
}
